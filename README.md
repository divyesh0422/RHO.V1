# RHO.V1

A modern web application built with React, TypeScript, and TailwindCSS, focused on performance, scalability, and clean UI/UX.

📌 Overview

RHO.V1 is a scalable front-end project designed to demonstrate modern development practices such as:

Component-based architecture
Optimized performance
Clean UI/UX design
Maintainable and reusable code

🛠️ Tech Stack
⚛️ React.js
🟦 TypeScript
🎨 TailwindCSS
⚡ Vite
🧪 Vitest (Testing)
✨ Features
⚡ Fast and optimized UI
🎯 Clean component structure
🌙 Dark mode support (if added)
🔍 SEO improvements
📦 Lazy loading for performance
❌ API error handling improvements

🚀 Getting Started
1. Clone the repository
git clone https://github.com/divyesh0422/rho-v1.git
cd rho-v1
2. Install dependencies
npm install
3. Run development server
npm run dev
🧪 Run Tests
npm run test
📦 Build for Production
npm run build
